# ReactQuiz

ReactQuiz is a simple and interactive quiz application built with React. The app presents users with a series of questions, tracks their answers, and displays a summary of their performance at the end.

## Features

- Multiple-choice questions with shuffled answers
- Timed questions and progress bar
- Immediate feedback on correct and incorrect answers
- Summary screen with statistics and answer review

## Getting Started

1. **Install dependencies:**
   ```
   npm install
   ```

2. **Run the development server:**
   ```
   npm run dev
   ```

3. **Open your browser:**  
   Visit [http://localhost:5173](http://localhost:5173) (or the port shown in your terminal).

## Project Structure

- `src/components/` – React components for quiz logic and UI
- `src/data/` – Questions data
- `src/assets/` – Images and static assets
- `src/index.css` – Styling

## About

This project was created as part of a React course to practice component-based development, state management, and user interaction in React.
